# SuuqLink Multi-Vendor Marketplace

SuuqLink is a complete, production-ready Multi-Vendor E-Commerce Marketplace built using vanilla HTML5, CSS3, and JavaScript (ES Modules) on the frontend, and Supabase (PostgreSQL, Authentication, and Storage) on the backend.

## 🚀 Features

*   **Bilingual Support:** Complete localization for English 🇬🇧 and Somali 🇸🇴.
*   **Theme Switcher:** Clean Light and Dark modes.
*   **User Roles:** Separate capabilities for Customers, Sellers/Merchants, and Delivery Drivers.
*   **Manual Payments:** Complete checkout integration supporting local manual mobile payment methods (EVC Plus, Golis Sahal, eDahab) with payment receipt verification.
*   **Driver Logistics:** Dispatch center for admins to assign drivers, and driver dashboard to manage delivery status.
*   **Seller Panel:** Dashboard showing metrics, weekly CSS bar chart, and product/inventory addition modal.
*   **Admin Console:** Management hub for manual receipt approvals, driver dispatcher, and system parameters.

## 🛠️ Database Setup

Run the migrations in `supabase/migrations.sql` inside the Supabase SQL editor to create all the database tables, views, RLS policies, storage buckets, and seed data.
